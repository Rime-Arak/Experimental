# Experimental - Target Practice

## Overview
I built this simple experiment entirely from scratch to test my independent problem-solving skills in Unity. It is a fundamental game loop demonstrating player mechanics, object spawning, and collision detection.

## The Game
* The player is restricted to side-to-side movement with arrow keys or A/D buttons.
* The player can fire projectiles at a moving enemy target with the space button.
* The enemy target moves side-to-side. When struck by a projectile, the target is destroyed, and a new one spawns to continue the loop.

* I've practiced with tags and expanded my knowledge on spawning methods.
